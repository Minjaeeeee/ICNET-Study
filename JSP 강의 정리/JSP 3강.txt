JSP 문서 작성하기
(1) JSP 특징
  - 동적 웹 어플리케이션 컴포넌트(HTML은 정적[리액션 X])
  - .JSP 확장자
  - JSP는 서블릿으로 변환되어 실행된다
  - MVC패턴에서 View로 이용된다. (Model View Controller[서블릿] )

(2) 웹 실행방식
  클라이언트가 주소url를 입력 -> MVC패턴이 작동된다-> 컨트롤러는 클라이언트가 무엇을 요청했는지 파악
  -> 모델한테 요청을 보내고 열심히 가공을하게 됨(DB에서는 데이터를 여러개 가지고 오고 등등) 
  -> 작업을 다하면 다시 컨트롤러에게 전달을 하고 사용자에게 응답하기위해 VIEW로 보냄
  -> 뷰는 클라이언트에게 응답을 해주면 홈페이지가 화면에 출력!

(3) Jsp 아키텍처
   - jsp파일을 컴파일하면 java파일로 변환(톰캣서버에서 변환) -> 클래스 파일로 변환해서 이 컴퓨터가 인식해서 브라우저로 응답
   - 클래스파일은 아파치 파일에 work 들어가서 쭉들어가면 클래스 파일이 존재한다.(사진 첨부하자) [21.39분]
